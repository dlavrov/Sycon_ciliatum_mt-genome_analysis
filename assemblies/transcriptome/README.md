# README.md
This directory contains new assemblies of transcriptomic data for *Sycon ciliatum*

## Assemblies of ERR12318596 dataset
- ERR12318596_3M_rnaSpades_raw.fasta.gz
- ERR12318596_rnaSpades_raw.fasta.gz
- ERR12318596_rnaSpades_trimmomatic.fasta.gz
- ERR12318596_trinity_raw.fasta.gz
- ERR12318596_trinity_trimmomatic.fasta.gz

## Other files
- previous_studies: a folder with assemblies of previously published datasets
- published_mt-cds.fa: mitochondial cds (transcripts) published in our previous study
- README.md: this file
- transcriptome_blast_db: directory with blast databases based on ERR12318596 assemblies
